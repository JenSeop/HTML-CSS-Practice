CSS style 3

인라인 스타일(inline style)
HTML 태그마다 style 속성으로 CSS 코드를 넣어주는 방식.
여러 요소들에 공통 속성 부여 불가능.
HTML 과 CSS의 분리가 불가능.
권장되지 않음.

내부 스타일 시트(internal style sheet)
head 태그 안에 style 태그를 두고 그 안에 CSS 코드 작성하는 방식.
HTML과 CSS의 전체 코드량이 많지 않고 CSS가 해당 HTML 문서에만 적용될 경우 사용.

링킹 스타일 시트(linking style sheet)
외부 CSS 파일을 HTML 문서에 연결하는 것.
HTML과 CSS의 코드가 분리되고 CSS 코드를 여러 HTML 파일에서 공통으로 사용 할 수 있음.
